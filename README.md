# On-Premise Data Pipeline for IoT & Weather Data

This project implements a full-stack data pipeline for processing weather and IoT data using various big data technologies.

## Technologies Used
- Apache Kafka
- Apache Spark (Streaming & Batch)
- Apache Airflow
- Apache Hive
- MySQL
- Docker
- Grafana + Prometheus
